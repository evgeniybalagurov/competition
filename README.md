# Проект в рамках проведения конкурса

Темой данного проекта является создание приложения, адаптированного для различных устройств.

## Технологии

- HTML
- CSS
- JavaScript

### Функционал:
- Поиск гифок
- Отображение популярных гифок
- Добавление своей гифки
- Выдача случайной гифки
- Валидация формы
